# PHP E-Commerce Project

This is a **PHP-based eCommerce project** with full CRUD (Create, Read, Update, Delete) functionality. Itâ€™s built using **PHP, MySQL, HTML, CSS, and JavaScript**.

## Features
- **Products:** Add, update, delete, and list products.
- **Orders:** View, update, and manage customer orders.
- **Customers:** Manage customer information and accounts.
- **Messages:** Send and view messages from customers.
- **Admin Panel:** Control all CRUD operations from a simple admin dashboard.

## How to Run
1. Install **XAMPP/WAMP/Laragon** on your computer.
2. Copy the project folder into the `htdocs` (or www) directory.
3. Import `database.sql` into **phpMyAdmin** to create the database.
4. Open `http://localhost/your-project-folder` in your browser.
